MyNotes - Your Complete Online Notes Library

MyNotes is an open source project which delivers to you the best-in class notes for programming and development. There are multiple languages and frameworks support and we wish to grow the community and reach in future.

Explore Resources

C

C is a procedural programming language, developed by Dennis Ritchie in 1972.

C++

C++ is an Object Oriented language, developed by Bjarne Stroustrup in 1985. It is the first choice when it comes to speed.

Python

Python is an interpreted language, created by Guido Van Rossum in 1991. It is easy to learn and is gaining popularity.

HTML

HTML is a markup language which is used to write content in websites. It basically defines the layout of a website.

CSS

CSS (Cascading Style Sheets) is used to style the HTML layout so that a website looks pretty.

JavaScript

JavaScript is a programming language which is used to add behaviour to a website, it defines the logic for it.

Dart

Dart is a language created by Google to support a framework called Flutter which is used to create cross-platform apps.

Java

Java is an Object Oriented language maintained by Oracle. It is a wide spread language and can be extended to app and web development

Kotlin

Kotlin was created by JetBrains in 2011 and has replaced Java as the official language for Android Development.

Git

Git is a very popular Version control system(VCS) which most programmers and companies use to keep track of the files in their projects.

Flutter

Flutter is a framework which was developed by Google in 2018 and is used to create cross-platform applications.

Vim

Vim is a modal text editor which was made by Bram Moonelaar in 1991. It is superfast and highly productive to use.

Linux

Linux is an Operating System developed by Linus Torvalds in the early 1990s. It is programmer friendly and gives you more reach into your system.
